# electropy
## Entropic Circuits

A minimalist real-time circuit maintenance game where you fight extreme entropy to keep the system alive.

You gradually learn the "language" of chaotic electronics: resistors dampen decay, capacitors store bursts of entropy while you reroute, diodes enforce flow direction, and transistors create satisfying chain-reaction relief moments.

**Core Loop**  
Drag components onto nodes/wires → manage rising entropy in real time → survive as long as possible. Every run teaches you better timing and placement strategies.

## Features (Current Proto)
- Real-time entropy simulation on circuit nodes
- 4 starter components:
  - **Resistor** – slows entropy growth
  - **Capacitor** – absorbs high bursts (emergency relief)
  - **Diode** – one-way flow stabilization
  - **Transistor** – threshold-triggered dumps for "aha" moments
- Simple drag-and-drop placement from palette
- Visual feedback: nodes glow redder as entropy rises
- Survival timer + overload failure state
- Ready for procedural level curation (50–100 hand-picked levels planned)
